Why We Built This

BlockDAG is a fundamentally different chain. It deserved mining software that understands that difference.

⚡

Custom Hash Engine

Not a copy-paste from Litecoin. Our Scrypt implementation (N=1024, r=1, p=1) includes BlockDAG's proprietary post-ROMix transformation — the exact algorithm the network validates. Written in pure C for maximum throughput.

🛠

Hardware-Aware

The installer detects your CPU architecture, core count, L2/L3 cache, and NVIDIA GPU. It auto-configures thread count, compiles natively for your chip (x86, ARM64, Apple Silicon), and offers CPU + GPU mixed-mode mining.

📊

Built-in Dashboard

Real-time web dashboard at localhost:8881 — hashrate, accepted shares, pool connection, per-thread stats. No third-party monitoring needed. Just open your browser.

🔒

Full Stratum Protocol

Complete stratum v1 implementation with vardiff support. Auto-reconnection, clean job switching, nonce range management. Compatible with any BlockDAG mining pool.

🏠

One-Command Install

Clone, run installer, mine. Three commands. Under sixty seconds. Dependencies, compilation, configuration, PATH setup, systemd service, desktop shortcuts — all handled. Zero manual config.

🚀

Battle-Tested

Running on our own infrastructure in production — Ras Al Khaimah, Cape Town, and community miners worldwide. Thousands of accepted shares. Real blocks. Real earnings. Not a testnet toy.
