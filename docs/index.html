From zero to mining in under 60 seconds.

Linux Installation Guide

1

Prerequisites

You need git installed. Most Linux distros include it. If not:

sudo apt install git # Debian/Ubuntu
2

Clone the Repository

Download the miner source code from GitHub:

git clone https://github.com/Dagtechltd/dagtech-miner.git && cd dagtech-miner
3

Run the Installer

The installer checks your hardware, installs dependencies (gcc, OpenSSL), then walks you through configuration:

chmod +x install.sh && ./install.sh
4

Configuration Wizard

The wizard asks for your wallet address, lets you choose CPU or GPU mode, select thread count, and set your pool. Default pool: excalibur.dagtech.network:3334

5

Start Mining

After installation completes, start mining with one command:

dagtech-start

Other commands: dagtech-stop to stop, dagtech-status to check stats. Dashboard at http://localhost:8881

macOS Installation Guide

1

Prerequisites

You need git (comes with macOS) and a terminal. Open Terminal.app from Applications > Utilities.

2

Clone the Repository

git clone https://github.com/Dagtechltd/dagtech-miner.git && cd dagtech-miner
3

Run the macOS Installer

Installs Xcode Command Line Tools if needed, detects Apple Silicon vs Intel, finds Homebrew OpenSSL:

chmod +x install-mac.sh && ./install-mac.sh
4

Configuration

Enter your wallet address when prompted. macOS uses CPU mining (optimized for Apple Silicon). The installer detects your M-series chip and sets optimal thread count automatically.

5

Start Mining

dagtech-start

Optional: The installer offers to create a LaunchAgent for auto-start on login. Your Mac will mine automatically every time you log in.

Windows Installation Guide

1

Prerequisites

Install Git for Windows from git-scm.com. Open Command Prompt or PowerShell as Administrator.

2

Clone the Repository

git clone https://github.com/Dagtechltd/dagtech-miner.git
cd dagtech-miner
3

Run the Installer

The batch installer automatically uses the pre-built binary from the repo. If a C compiler is available, it can optionally build from source:

install.bat
4

Configuration

Enter your wallet address, choose GPU/CPU mode. The installer detects NVIDIA GPUs via WMIC and offers GPU-accelerated mining if available.

5

Start Mining

dagtech-start.bat

Other scripts: dagtech-stop.bat and dagtech-status.bat. Dashboard at http://localhost:8881

static void dagtech_hash(const uint8_t *input, uint8_t *output) {
  /* Modified Scrypt with post-ROMix transform */
  uint8_t B[128];
  uint32_t *V = malloc(N * 128);

  PBKDF2_SHA256(input, 80, input, 80, 1, B, 128);
  scrypt_ROMix(B, V, N);
  dagtech_post_transform(B, 128);
  PBKDF2_SHA256(input, 80, B, 128, 1, output, 32);

  free(V);
}
