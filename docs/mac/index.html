v0.1 - 2026-06-19 - Apple Silicon (arm64)

dagtech-mac-miner

One-liner CPU miner for BlockDAG on Apple Silicon Macs.

Install
curl -fsSL https://miner.dagtech.network/mac/install.sh | bash

What happens when you run it

The installer is designed for a clean Mac with nothing pre-installed. It does all of the following without you touching anything else:

  • Detects macOS + Apple Silicon - bails politely on Intel
  • Installs Xcode Command Line Tools if missing (Apple's official downloader)
  • Installs Homebrew if missing (official curl|bash from brew.sh)
  • Installs openssl@3 via Homebrew if missing
  • Downloads the verified binary (SHA256 chain - primary + fallback URLs)
  • Ad-hoc codesigns the binary so Gatekeeper allows it
  • Asks for your wallet, worker name, threads (or accepts DAGTECH_* env vars)
  • Writes config to ~/.dagtech-miner/config.json
  • Registers two launchd services that survive reboot and crash
  • Opens the live dashboard at http://127.0.0.1:8765

You do not need an Apple Developer account, do not need to grant manual permissions, do not need to know Terminal.

System requirements

OS

macOS 12 Monterey or later. Recommended macOS 14 Sonoma.

CPU

Apple Silicon - M1, M1 Pro, M2, M3 or later. Intel not supported in v0.1.

RAM

8 GB minimum, 16 GB recommended.

Disk

200 MB free.

Verifying the binary

The installer verifies the SHA256 automatically. To verify manually after install:

shasum -a 256 ~/.dagtech-miner/bin/dagtech-mac-miner-cpu
expected: c9222f7e022ab06c17d785ca44d737bd7580391c95e76ee5748ef06a71c202bf

Reproducibility

v0.1 was validated against a 10-iteration install/verify/uninstall reproducibility loop on a clean M1 Pro running macOS 26.5. Each pass tears down completely, runs the installer non-interactively, waits 8 seconds, then verifies launchctl PIDs are alive, dashboard responds, miner.log shows mining activity, and dashboard worker name matches the per-iteration config.

REPRODUCIBILITY SCORE: 10/10 passed, 0/10 failed

Source
